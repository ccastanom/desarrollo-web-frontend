# Proyecto Angular con TypeScript

Este proyecto es una aplicación de gestión de usuarios creada con Angular, TypeScript, y Node.js , la cual permite crear, listar, editar y eliminar usuarios con autenticación por roles.

## Requisitos

- Node.js (versión 18 o superior)
- npm (versión 10 o superior)

## 🛠️ Tecnologías usadas

- Angular 16
- PrimeNG / Angular Material
- Node.js + Express
- PostgreSQL
- Sequelize ORM
- JWT para autenticación

## ⚙️ Funcionalidades

- Login de usuarios (admin y usuarios normales)
- Crear nuevo usuario
- Editar información de usuario
- Eliminar usuario
- Asignación de administrador (si el rol es usuario)

## 🚀 Instalación
### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
Copiar
cd desarrollo-web-frontend
npm install
ng serve
```
